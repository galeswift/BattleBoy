Vista Users:

 Downloaded .chm files are blocked in Vista by default, there are some work arounds an the microsoft knowledebase.
 In short: Right click the .chm file, open the properties tab and click 'unblock'.
 Alternatively, you can browse the documentation online at http://www.ambiera.com/irrklang



Linux / MacOS Users:

 There are some tools at the web for reading .chm files (try http://xchm.sourceforge.net/).
 You can find a html version of this documentation at http://www.ambiera.com/irrklang
